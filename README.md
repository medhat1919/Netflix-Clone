# 🎬 Netflix Clone - React & Vite

A high-performance, responsive Netflix clone built with React, Vite, and Tailwind CSS. This project leverages the TMDB API to fetch real-time movie and TV show data, providing a premium streaming experience.

🚀 **Live Demo:** [https://medhat1919.github.io/Netflix-Clone/](https://medhat1919.github.io/Netflix-Clone/)

## ✨ Features

- **Instant Access:** Bypasses login to go straight to the content after a cinematic splash screen.
- **Dynamic Content:** Fetches trending, popular, and top-rated movies/TV shows from TMDB.
- **Responsive Design:** Optimized for all devices, including mobile and high-definition screens.
- **Movie Details:** Modal view for expanded information, trailers, and metadata.
- **Personalized List:** Add and manage your favorite titles in "My List".
- **Real-time Search:** Quickly find content across the entire library.
- **Trailers:** Seamlessly play YouTube trailers directly within the app.

## 🛠️ Technology Stack

- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS 4
- **Routing:** React Router 7 (HashRouter for GitHub Pages)
- **Animations:** Framer Motion & GSAP
- **Icons:** Lucide React
- **API:** TMDB (The Movie Database)

## 📦 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/medhat1919/Netflix-Clone.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create a `.env` file and add your TMDB token:
   ```env
   VITE_TMDB_TOKEN=your_token_here
   ```

4. **Run development server**
   ```bash
   npm run dev
   ```
